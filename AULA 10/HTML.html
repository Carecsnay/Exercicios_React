Abordagem sobre a tag select (Link Externo)

A tag oferece uma espécie de menu de opções. As opções são encapsuladas pelo elemento < option> item < / option>

Outro tipo de select é o select multiplo, na qual você segura ctrl e vai clicando no item desejado
