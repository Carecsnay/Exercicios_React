Padding -> preenchimento interno e contabiliza também no tamanho total do elemento. Ex.: no vídeo acima é colocado um elemento de tamanho 400px, porem ao inspecionar o tamanho passa a ser 430px pois existe o padding de 10 + 10 = 20 e border de 5 + 5 = 10. Para contornar o problema, no css deve colocar no seletor * para aplicar em todos os elementos * { box-sizing: content-box; (o tamanho do elemento é definido pelo conteúdo, isso não inclui os padding e border) box-sizing: border-box; (aqui vai incluir tanto o padding e o border, vão ser removido dos 400px para ficar do tamanho definido la em cima 400px) } Margin -> preenchimento externo e não contabiliza no tamanho do elemento. No CSS se colocar asterisco, o efeito será aplicado em todos os elementos
