Leitura do CSS

Folhas de estilo em cascata.

Como mencionado na aula anterior, o CSS faz a leitura de cima para baixo, portanto a última propriedade aplicada será aplicada (caso o elemento não seja especificado - id, classe).

Se colocarmos o CSS diretamente no HTML (CSS-inline), o efeito seria aplicado, pois o HTML por padrão lê primeiramente o CSS externo e só depois ele verifica o CSS aplicado no HTML (CSS-inline). Sendo assim, seguindo o princípio Cascata
