Margins -> Espaçamento EXTERNO Padding -> Espaçamento INTERNO 4 propriedades distintas que todo ELEMENTO tem (você pode conferir inspecionando um elemento. - Conteúdo - Borda - Padding - Margin Podemos tirar o espaçamento do (em cima da página) No CSS body { margin: 0px; } Você também pode alterar dimensões de tamanhos do Padding, Margin seguindo o seguinte exemplo: No CSS .texto { margin: 1px 5px 1px 5px; // na qual o primeiro valor equivale em cima, o seguindo a direita, terceiro baixo e 4 } esquerda. Ordem Horária. também podemos reescrever o código da seguinte maneira .texto { margin: 1px 5px; //já que os valores são iguais cima/baixo e lado/outro. }
