RESUMO


Abordagem sobre tipos de input no HTML5 (Navegadores Antigos não são suportados!)

Input Color -> Input de seleção de cor (Paleta de Cores).

Input Date -> Input de seleção de data (Input de Datas) o dado é enviado no formato internacional aaaa-mm-dd.

Input Datetime-local -> Input de seleção de data (Input de Datas com Tempo) o dado é enviado no formato internacional aaaa-mm-dd e inclui também o tempo.

Input Month -> Input de seleção de data (Input de Mês).

Input Time -> Input de seleção de tempo (Input de Tempo).

Input Week -> Input de seleção de semanas (Input de Semana).

Input Email -> Input de inserção de email (Input de E-mail).

Input Number -> Input de inserção de email (Input de Números).

Input Range -> O usuário especifica um valor numérico que não deve ser inferior a um determinado valor (Input Range).

Input Search -> Insere Input de pesquisa (Input de Pesquisa).

Input Tel -> Insere input de telefone, apesar de não formatar (Input de Telefone).

Input URL -> Insere input de URL(Input de URL).
