FORMULÁRIOS

Representa uma seção de um documento que contém controles interativos que permitem ao usuário submeter informação a um determinado servidor web.

A tag form recebe 1 atributo e 1 método na qual foi ensinado a usar o atributo action que por sua vez requisita o endereço para onde você vai enviar as informações preenchidas no formulário e o método post que é mais usado para postar os dados do formulário, já que os dados são enviados internamente, portanto o usuário não tem acesso a eles.

Exemplo de código

ÁREA DE CADASTRO

< form action="...endereço a onde você vai enviar as informações" method="post">
