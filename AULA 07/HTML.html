Formas de alterar a cor no CSS Modo 1 (cor em inglês) #tittle { background-color: red; } Modo 2 (usando a paleta rgb 0 a 255(por causa q é em bytes 255=100% e 0=0%)) #title { background-color: rgb(0,0,0); } Modo 3 (usando código hexadecimal) Most Recommended! #tittle { background-color:#FF00FF ou #F0F (resumida cor geral); }
