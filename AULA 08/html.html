Inherit (Forçar Herança)

Basicamente o inherit força uma herança de determinadas propriedades.

Coisas herdáveis (exemplos):
Color
Font-size

Exercício de Fixação !
Para que é usado o inherit no HTML&CSS ?

1 Para herdar propriedades do elemento em que ele está contido
2 Para mudar a cor do elemento principal da página
3 Para salvar um tipo de propriedade para todos os elementos de uma página
4 Todas as alternativas estão incorretas
